# Student Leave Management Web App

Tech Stack: Python Flask + HTML + CSS + SQLite

Features:
• Student Leave Applying Portal
• Advisor Approve/Reject
• HOD View Only
• Attendance Tracker (25% rule)
• Formal leave letter generation

Roles:
• Student
• Class Advisor
• HOD

Setup:
1. python database_setup.py
2. python app.py
Login Sample:

advisor@example.com / advisor123
hod@example.com / hod123
